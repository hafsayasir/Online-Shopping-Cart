# Online-Shopping-Cart
A Python CLI app that simulates fast-food ordering. It provides secure sign-up/login, a dynamic menu with realtime prices, and a cartbased order system. User data and order history are saved using file handling. Programming concepts include dictionaries, lists, file I/O, exception handling and control flow to ensure maintainable experience.
Features

User Authentication: Secure sign-up and login system.

Dynamic Menu: Interactive menu with live pricing.

Order Management: Add items to cart and calculate totals.

Data Persistence: Stores user data and order history locally.

Technical Concepts

File Handling: Read/write .txt files for data storage.

Data Structures: Dictionaries for menus and profiles, lists for orders.

Exception Handling: Handles errors to ensure smooth execution.

Modular Programming: Organized code using helper modules.

Control Flow: Loops and conditionals for input validation and process flow.
